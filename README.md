# FANTASY MEMO App

HTML/CSS/JavaScriptで作成した、シンプルで使いやすいメモアプリケーションです。

## 🚀 公開URL
[https://t226t091-ship-it.github.io/my-app-2026/](https://t226t091-ship-it.github.io/my-app-2026/)

## ✨ 主な機能
- **メモの追加**: タイトル（任意）と内容を入力して保存できます。
- **メモの編集**: 一度保存したメモを後から修正・更新できます。
- **自動保存**: ブラウザの `localStorage` を使用しているため、ページを閉じてもメモが保持されます。
- **検索機能**: メモのタイトルや内容からリアルタイムに検索・絞り込みが可能です。
- **削除機能**: 不要になったメモを簡単に削除できます。
- **レスポンシブデザイン**: PC、タブレット、スマートフォンなど、あらゆるデバイスで最適に表示されます。

## 📖 使い方
1. **メモを追加する**:
   - 画面中央の入力エリアにタイトルとメモの内容を入力します。
   - 「追加」ボタンをクリックすると、下のリストにメモが表示されます。
2. **メモを編集する**:
   - 修正したいメモの「編集」ボタンをクリックします。
   - 入力エリアに内容が表示されるので、修正して「更新」ボタンをクリックします。
   - 編集を中断したい場合は「キャンセル」ボタンをクリックします。
3. **メモを検索する**:
   - 画面上部の検索バーにキーワードを入力すると、一致するメモだけが表示されます。
4. **メモを削除する**:
   - 各メモカードの右下にある「削除」ボタンをクリックし、確認ダイアログで「OK」を選択します。

## 🛠 技術スタック
- **HTML5**: アプリの構造定義
- **CSS3**: モダンなデザインとアニメーション
- **JavaScript (ES6+)**: メモの管理ロジックとデータ保存
- **localStorage**: ブラウザでのデータ永続化

## 📝 開発者向け
ビルド工程は不要です。`index.html` をブラウザで開くだけですぐに動作確認が可能です。
